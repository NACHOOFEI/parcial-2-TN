Rick and Morty API

Esta API te permite acceder a datos de los personajes, ubicaciones y episodios de la serie Rick and Morty.

¿Qué podés hacer?

  • Buscar personajes por nombre
  • Filtrar por estado: vivos, muertos o desconocidos
  • Ordenar alfabéticamente
  • Ver imágenes y estado de cada personaje
